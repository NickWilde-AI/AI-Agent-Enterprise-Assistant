# 用户流程图（文字版）

```text
用户提出问题/任务
        |
        v
意图识别：
- 仅问知识？ -> 走 RAG
- 需要数据？ -> 调 MCP 工具
- 需要生成材料？ -> RAG + MCP + 生成草稿
        |
        v
系统执行：
1) 检索知识库（RAG） -> 返回制度/标准依据
2) 调用工具（MCP） -> 查询员工/项目/报销数据
3) 生成结果 -> 邮件/工单/说明
        |
        v
用户确认/补充
        |
        v
系统输出最终结果（含来源）
```

## 关键流程（报销说明邮件）
```text
用户：提交“查询张三 2025-03 报销总额并写邮件”
  |
Agent：判断涉及金额 -> 必须调用 MCP
  |
MCP：get_reimbursement_summary -> 返回明细与总额
  |
Agent：生成邮件草稿 -> 提示用户确认
  |
用户：确认/补充 -> 输出最终邮件
```
